# Everest Security Website

Static HTML/CSS/JS website for Everest Security.

## Structure

```
.
├── index.html          # Main entry point
├── 404.html           # SPA redirect for GitHub Pages
├── assets/
│   ├── index-Bs0mItmB.js    # Compiled JavaScript
│   └── index-B57ClfSU.css   # Compiled CSS
└── README.md
```

## Deployment

This site is configured for GitHub Pages. To deploy:

1. Push these files to your repository
2. Enable GitHub Pages in repository settings
3. Set source to deploy from the branch containing these files

## Features

- Pure static files (no build step required)
- SPA routing support via 404.html redirect
- Responsive design
- All pages work on refresh/direct links
